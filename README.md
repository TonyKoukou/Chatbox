# Mini Chatbot


## Περιγραφή
Python πρόγραμμα που λειτουργεί σαν απλός chatbot. Απαντά σε ερωτήσεις όπως "hello", "python", ή "bye".


## Οδηγίες Χρήσης
1. Βεβαιώσου ότι έχεις εγκατεστημένη Python 3.
2. Άνοιξε το πρόγραμμα:cmd
3. Γράψε cd και την τοποθεσία του αρχείου π.χ cd C:\Users\Tony\MiniChatbox
4. Γράψε την εντολή: python chatbox.py
5. Πληκτρολόγησε ερωτήσεις. 
6. Πληκτρολόγησε "bye" για να κλείσεις.
